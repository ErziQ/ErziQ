## ErziQ Website

A modern, highly polished website with Home and Settings tabs, showcasing maximum UX/UI design excellence. The Home tab features interactive text fields with smooth animations and clipboard functionality, while Settings offers customizable options.

### ✨ Features
- **Tab Navigation**: Smooth tab switching with icons and hover effects
- **Interactive Text Fields**: Click to copy prompts with enhanced animations and visual feedback
- **Dark Mode**: Toggle between light and dark themes with persistent preferences
- **Copy Notifications**: Optional floating notifications when copying text
- **Modern Design**: Gradient backgrounds, glassmorphism effects, and responsive layout
- **Smooth Animations**: Fade-ins, slide transitions, hover effects, and copy animations
- **Accessibility**: Proper contrast, keyboard navigation, and screen reader support

### 🎨 UI/UX Highlights
- **Glassmorphism**: Backdrop blur effects and translucent elements
- **Micro-interactions**: Hover animations, button presses, and state changes
- **Color System**: CSS custom properties for consistent theming
- **Typography**: Modern font stack with proper hierarchy
- **Responsive Grid**: Adaptive layout that works on all devices
- **Loading Animation**: Smooth page entrance effects

### 🚀 How to Run
1. Open `index.html` directly in your web browser for instant viewing
2. Or serve with a local HTTP server:
   ```bash
   python3 -m http.server 8000
   ```
   Then visit `http://localhost:8000`

### 📁 Files
- `index.html`: Main HTML structure with Font Awesome icons
- `styles.css`: Advanced CSS with CSS variables, animations, and responsive design
- `script.js`: JavaScript for interactions, dark mode, and local storage

### 🛠️ Technologies Used
- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons
- Local Storage API
